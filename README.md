📖 Descripción del proyecto

Este proyecto es una calculadora desarrollada en Java, diseñada para ejecutarse en consola.
Su objetivo es ofrecer una herramienta sencilla pero bien estructurada, aplicando buenas prácticas de programación, validación de entrada y un menú interactivo.
El proyecto se encuentra en desarrollo continuo y servirá como base para futuras versiones con interfaz gráfica y conexión a base de datos.

⚙️ Funcionalidades

✅ Suma de dos números

✅ Resta de dos números

✅ Multiplicación (nueva en la versión 1.1)

✅ División (nueva en la versión 1.1)

⚠️ Validación de entrada: se asegura que el usuario solo ingrese valores numéricos

🧾 Menú interactivo: permite seleccionar operaciones de forma intuitiva

🚀 Cómo ejecutar el proyecto
Opción 1: Desde Apache NetBeans

Clonar el repositorio:

git clone https://github.com/Rolfata/calculadora.git


Abrir el proyecto en Apache NetBeans.

Ejecutar el archivo principal:
src/Calculadora/Calculadora_app.java

Opción 2: Desde la terminal (compilación manual)

Abrir la carpeta del proyecto.

Compilar el archivo principal:

javac src/Calculadora/Calculadora_app.java


Ejecutar el programa:

java src.Calculadora.Calculadora_app
